Il progetto di ricerca che sta alla base di questo sito è il tentativo di raccogliere tutte le evidenze, documentarie e iconografiche, relative a Gracia Nasi alias Beatriz de Luna (1510-1569), donna di straordinaria influenza nella storia degli ebrei sefarditi in età moderna.
L'obiettivo è quello di rendere disponibili gli oggetti digitali per la consultazione, che può essere fatta sia attraverso un catalogo filtrabile per tipologia di fonte, sia attraverso metodi di accesso specifici quali un indice delle persone, una mappa cartografica e una linea del tempo.
Gli oggetti raccolti nel sito sono tutti catalogati attraverso metadati descrittivi con standard DC e le fonti documentarie (manoscritti, lettere, documenti notarili) sono descritte attraverso lo standard TEI. Questo garantisce l'interoperabilità con altri sistemi e la riusabilità dei dati nel tempo.

Standard per i metadati

Dublin Core (DC) e Dublin Core Terms (DCTERMS) per la descrizione degli item, implementati sia nei tag <meta> dell'<head> HTML sia in un file XML dedicato (item_DC.xml). I record di autorità sono riconciliati con VIAF per le persone e GeoNames per i luoghi.

TEI (Text Encoding Initiative) per la codifica filologica delle fonti documentarie (item_TEI.xml), con marcatura di persone, luoghi, date, abbreviazioni e note archivistiche.
